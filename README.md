# Сокращение ссылок с помощью сервиса Битли

Консольная утилита для сокращения ссылок с помощью сервиса Bitly - http://bit.ly/
А также для просмотра количества переходов по уже сокращённой ссылке.

### Запуск:
- для сокращения ссылки передайте её в качестве аргумента
```
main.py  https://www.google.com
Битлинк - bit.ly/39LzxjH 
```
- для просмотра количества переходов по битлинк ссылке передайте её в качестве аргумента
```
main.py  bit.ly/39LzxjH
Всего переходов по ссылке - 1
```

### Как установить

Для того, чтобы пользоваться утилитой, необходимо получить ключ - "токен", для этого необходимо
зарегистрироваться в [Сервисе Bitly](https://bit.ly/) и сгенерировать [токен](https://bitly.com/a/oauth_apps).
Токен выглядить так:
```
17c09e20ad155405123ac1977542fecf00231da7
```
После этого нужно создать файл .env и в нём присвоить переменной BITLY_TOKEN полученный токен:
```
BITLY_TOKEN=17c09e20ad155405123ac1977542fecf00231da7
```
Python3 должен быть уже установлен. 
Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:
```
pip install -r requirements.txt
```
